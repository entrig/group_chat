# Lovable MCP setup

Lovable supports custom MCP servers as Chat connectors on paid plans.

Use Lovable's connector UI:

1. Open Connectors -> Chat connectors.
2. Click New MCP server.
3. Server name: `Entrig`.
4. Server URL: `https://mcp.entrig.com/beta`.
5. Authentication: choose Bearer token or API key. Do not use OAuth for Entrig.
6. Token/API key: paste the Entrig API key from dashboard project settings.
7. Click Add server.

After connecting, ask Lovable to create or manage the Entrig notification again so it can use the newly available MCP tools.

Lovable chat connectors are per-user connections. Users can review or revoke them from Connectors.
